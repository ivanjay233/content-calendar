# Changelog

## 0.1.0 (Unreleased)

### Features

- **Core:** CalendarManager with CRUD operations and persistent storage
- **Models:** ContentItem, ContentCalendar, enums for status/type/platform
- **Planner:** AI-powered topic generation with customizable templates
- **Exporter:** CSV, JSON, Markdown, HTML, and iCalendar formats
- **CLI:** Full command-line interface with init, plan, topics, export, stats
- **Config:** YAML-based configuration with content types and platform schedules

### Infrastructure

- Project scaffolding with pyproject.toml and setuptools
- Comprehensive test suite (25+ tests) with pytest
- GitHub Actions CI for Python 3.10, 3.11, 3.12
- Pre-commit hooks for ruff and mypy
- Makefile for common development tasks
